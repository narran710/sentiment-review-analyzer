# 🧠 Sentiment Review Analyzer  
A complete end-to-end NLP project for analyzing the sentiment of reviews using **BERT** and **HuggingFace Transformers**.  
Supports training, evaluation, and real-time inference.

---

## 📌 Features
- Binary Sentiment Classification (Positive / Negative)
- Train using HuggingFace Trainer
- Evaluate accuracy and F1 score
- Save & load the best model
- Simple inference script for predictions
- Clean modular project structure
